
## 安裝Docsify撰寫文件網站

```
npm i docsify-cli -g
```

run.bat

```
docsify serve ./ROOT
```

run.sh

```bash
#!/bin/bash
DIR=`pwd`
ROOT="$DIR/ROOT"
"$DIR"/docsify-cli/bin/docsify serve $ROOT --port 3000
```
